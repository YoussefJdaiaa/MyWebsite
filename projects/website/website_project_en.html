Project Overview

Complete development of a web platform for managing a sports association. The project includes designing a robust relational database and developing an intuitive web interface for comprehensive management of members, activities, and association events.

Technical Details

Technologies Used:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: PHP 7.4+
  • Database: MySQL / MariaDB
  • Web Server: Apache
  • Tools: phpMyAdmin for database management

Database Architecture:

  • Members Table: Personal information, contact details, membership status
  • Activities Table: Different sports and activities offered
  • Registrations Table: Link between members and activities
  • Events Table: Competitions, tournaments, meetings
  • Payments Table: Tracking membership fees and registration costs

Database Schema

Integrity Constraints:

  • Auto-incremented primary keys
  • Foreign keys with referential constraints
  • Format validation (email, phone, dates)
  • Optimized indexes for frequent searches

Implementation

Project Structure:

  • /config/ - Database configuration and settings
  • /includes/ - Reusable PHP files (header, footer, functions)
  • /admin/ - Administration interface
  • /members/ - Member area
  • /assets/ - CSS, JavaScript, images
  • /sql/ - Database creation and population scripts

Advanced Features:

  • Advanced search and filtering system
  • Data export in PDF and Excel formats
  • Responsive interface adapted for mobile/tablet
  • Automatic data backup system
  • Activity logs for audit and security
